# TS-PY
### Skole proskjet

#### Client setup
```
yarn install
```

##### Compiles and hot-reloads for development
```
yarn run serve
```

##### Compiles and minifies for production
```
yarn run build
```

##### Run your tests
```
yarn run test
```

##### Lints and fixes files
```
yarn run lint
```

#### Server Setup

Requirements:
  Python3
  Pip3
  Sqlite

##### Dependencies
```
pip install flask flask_sqlalchemy
```
##### Run server
```
python api.py
```
